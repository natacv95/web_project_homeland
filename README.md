# Galería de Arte TripleTen

## Descripción

Este proyecto es una galería de arte desarrollada como parte del curso de Frontend de TripleTen. Presenta imágenes y textos sobre distintos lugares del mundo utilizando una estructura modular de CSS basada en BEM.

## Funcionalidad

- Página principal con secciones de header, main, gallery, stories y footer.
- Diseño responsivo adaptable a diferentes tamaños de pantalla.
- Estilos organizados de forma modular para facilitar mantenimiento.

## Tecnologías y técnicas utilizadas

- HTML5
- CSS3 con metodología BEM (estructura plana)
- Responsive Design (media queries)
- Normalización con `normalize.css`
